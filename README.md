# Gary Ahuja PREC. Website

## How to preview
Open `index.html` in your browser.

## How to edit
- Main content: `index.html`
- Design/colors: `styles.css`
- Mobile menu/animations: `script.js`
- Images: `assets/`

## How to publish free with GitHub Pages
1. Create a GitHub account.
2. Create a new repository named `gary-ahuja-website`.
3. Upload all files from this folder.
4. Go to Settings → Pages.
5. Under Source, choose `Deploy from branch`, branch `main`, folder `/root`.
6. Connect your GoDaddy domain in Pages custom domain settings.

## GoDaddy DNS for GitHub Pages
Add these A records pointing to GitHub Pages:
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

For `www`, add a CNAME pointing to your GitHub Pages URL.
